# Console Perfect Log Client 

This is a simple console client for Perfect Log Server, 
which allows you to send log messages directly from the command line.

Contents Logic 

Save every detail of application inside Log Data instance. 
LogData instance will get 64KB size memory block to store log details. 

In advance, it might be implemented to save that 
64KB log instance into file system, when either 

1. LogData instance is full 
2. Program Shuts down
